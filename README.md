# Smart Study & Discipline Tracker

A Python-based CLI application to track daily study habits, calculate streaks, and generate weekly progress summaries.

## Why This Project Exists

Consistency is harder than motivation. This tool helps track real progress across learning, faith, and discipline.

## Features (MVP)

- Track daily habits
- Persistent storage
- Streak calculation
- Weekly summaries
- Command-line interface

## Habits Tracked

- Python study
- Korean study
- Qur'an memorization
- Reading
- Exercise

## Tech Stack

- Python 3
- JSON storage
- CLI-based interface
